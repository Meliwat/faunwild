# Code of Conduct

FAUNWILD is a game people build together for fun. Keep it that way:

- Be kind. Assume good faith. Critique code and ideas, never people.
- Welcome newcomers — many contributors here are making their first PR.
- No harassment, hate speech, or personal attacks, anywhere in the project.
- Keep content in the game suitable for all ages.
- Disagreements about design happen; settle them in issues with arguments,
  not in PR review battles. Maintainers make the final call.

Report problems privately to the maintainers. Violations can lead to warnings,
reverted contributions, or bans from the project.
